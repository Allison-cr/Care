Care

Приложение отслеживает геопозицию
С помощью weatherapi предоставляет данные погоды
Помимо геолокации и данных температуры снизу реализована CollectionView с примерами одежды
CollectionView предоставляет 10 картинок в зависимости от данных температуры
Если холодно, то предлагает зимние варианты одежды
Если тепло, то предлагает теплые варианты одежды
Также реализована смена фона главного экрана в зависимости от реального времени( днем светлый задний фон, вечером темный фон)
